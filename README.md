# nova → astro

This repo is **archived**. Functionality has been merged into
[astro](https://github.com/isene/astro), which combines nova's sky
panel with scope's telescope/eyepiece catalog into a single
amateur-astronomy app.

Existing config at `~/.nova/config.yml` is automatically migrated
to `~/.astro/config.yml` on first launch of astro.
